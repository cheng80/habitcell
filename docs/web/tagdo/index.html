태그로 분류하는 스마트 할 일 관리

할 일을 태그로 분류하고, 마감일 알림으로 놓치지 마세요.
심플하지만 강력한 할 일 관리 앱.

기능 살펴보기 다운로드

주요 기능

TagDo가 제공하는 핵심 기능을 확인하세요.

🏷️

태그 기반 분류

커스텀 태그와 색상으로 할 일을 직관적으로 분류하세요. 태그별 필터링으로 원하는 항목만 빠르게 확인할 수 있습니다.

⏰

마감일 알림

마감일을 설정하면 정확한 시간에 알림을 받을 수 있습니다. 기기 재시작 후에도 알림이 자동으로 복원됩니다.

🌙

다크 모드

라이트 / 다크 / 시스템 연동 3가지 테마를 지원합니다. 눈의 피로를 줄이고 배터리도 절약하세요.

🌏

다국어 지원

한국어, 영어, 일본어, 중국어(간체/번체) 5개 언어를 지원합니다. 시스템 언어에 맞춰 자동 적용됩니다.

🔒

완전한 프라이버시

모든 데이터는 기기에만 저장됩니다. 서버 전송 없이 완전한 오프라인으로 동작하여 개인정보가 안전합니다.

💡

직관적 UI

Material Design 기반의 깔끔한 인터페이스. 처음 사용해도 쉽게 익힐 수 있는 가이드 투어를 제공합니다.
