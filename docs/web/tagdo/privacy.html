1. Purpose of Processing Personal Information

TagDo (hereinafter "the App") processes personal information for the following purposes. Personal information being processed will not be used for purposes other than those listed below, and if the purpose of use changes, necessary measures such as obtaining separate consent will be implemented.

  • Providing todo management and tag-based categorization services
  • Providing due date notification (local notification) services
  • Saving app settings and improving user experience

2. Personal Information Collected

TagDo does not collect or transmit any personal information to servers.

All data is stored only on the user's device (locally) and is not transmitted to external servers.

3. Provision of Personal Information to Third Parties

TagDo does not provide users' personal information to third parties. All data is processed only within the device.

4. Retention and Usage Period

All data is stored on the user's device, so all data is deleted when the app is uninstalled.

  • Retention Period: During the app installation period
  • Deletion Method: Automatically deleted upon app uninstallation or data reset within the app

5. App Permissions

TagDo may request the following device permission to provide its services.

PermissionPurposeRequired
Notifications (POST_NOTIFICATIONS)Sending due date notificationsOptional

The notification permission is optional, but if not granted, due date reminder notifications will not be available. Core features such as creating, editing, and deleting todos work normally without this permission.
