1. Purpose

GlucoInsight processes information only as needed to provide features such as diabetes risk estimation, address storage, and nearby hospital search.

2. Data Processed

Depending on your use, the app may process:

  • Prediction inputs: sex, age, height/weight (BMI), waist circumference, optional fasting glucose, optional questionnaire answers
  • Address/location-related data: user-entered address and converted coordinates (lat/lng)
  • App settings: theme, API base URL, and related configuration values
  • Technical logs: minimal non-identifiable logs for stability and error diagnostics

3. Processing and Storage

  • Some data is stored locally on device storage (GetStorage).
  • Input data may be transmitted to APIs to provide prediction and geocoding features.
  • When opening map directions, coordinates may be passed to the map app selected by the user.

4. Third-Party Services

We do not sell personal data. External services may be used only to deliver app functionality:

  • Prediction API server for risk estimation
  • Address/map-related services for search, geocoding, and navigation
  • App store review flow when users choose to leave a review

Each external service processes data under its own policy.

5. Retention

  • Locally stored data may be removed when users edit/delete data or uninstall the app.
  • Server-side retention, if any, follows server operation policy and applicable law.
  • Data required by law may be retained for the legally required period.

6. User Rights

  • Users can modify input values, update address data, and remove local data by uninstalling the app.
  • For privacy requests, contact us at the email below.

7. Permissions

The app uses permissions within the minimum scope required for functionality.

  • Network access: prediction API and hospital search API communication
  • Optional features: address/map search and app review prompt flow

Core prediction features are available even if optional features are not used.

8. Children's Privacy

This app is designed for adults and is not intended to knowingly collect children's personal data.

9. Policy Updates

This policy may be updated due to legal or service changes. Updates will be posted on this page.
